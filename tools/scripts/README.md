# Runtime Scripts

Scripts for running and managing Odyssey ERP application.

## Available Scripts

- **run.sh** - Run application in foreground (interactive mode)
- **run-background.sh** - Run application in background (daemon mode)
- **status.sh** - Check application status
- **stop.sh** - Stop running application
- **SETUP_COMMANDS.sh** - Initial setup commands

## Usage Examples

```bash
# Start application
./run-background.sh

# Check status
./status.sh

# Stop application
./stop.sh
```

For detailed usage, see: `documentation/GETTING_STARTED.md`
