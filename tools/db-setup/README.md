# Database Setup Scripts

Scripts for initializing and setting up PostgreSQL database.

## Available Scripts

- **setup-db.sh** - Main database setup script (recommended)
- **setup-db-simple.sh** - Simplified setup script
- **setup-db-manual.sh** - Manual setup with prompts
- **create-db-interactive.sh** - Interactive database creation
- **create-database.sql** - SQL script for database creation
- **fix-postgres-and-setup.sh** - Fix PostgreSQL issues and setup

## Quick Start

```bash
# Run main setup script
./setup-db.sh
```
